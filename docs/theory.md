# Floating Point Numbers Representation
A simple representation of a floating-point (or real) number (N) uses a fraction (F), base (B), and exponent (E), where `N=Fx(B^E)`. The base can be 2, 10, 16, or any other integer larger than 1. The fraction and the exponent can be represented in many formats. 

**For example**, they can be represented by 2’s complement formats, sign-magnitude form, or another negative number representation.
